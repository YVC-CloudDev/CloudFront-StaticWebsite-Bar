Are the courses really free?

Yes! All core course materials are completely free and open source. We believe education should be accessible to everyone.

Do I need prior experience with CI/CD?

Not at all. Our beginner courses start from the very basics. You just need to know basic Git commands.

Can I use these projects in my portfolio?

Absolutely! Fork any project, customize it, and showcase it. That's exactly what they're designed for.

How do I deploy this site with GitHub Actions?

Fork this repository, enable GitHub Pages in your repo settings, and the included workflow will handle the rest automatically!
